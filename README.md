## ⚠️ Things to Modify Per Project:

| Variable | Description |
|---|---|
| `<your-nexus-ip>` | Your Nexus server IP |
| `your-cluster-name` | Your EKS cluster name |
| `us-east-1` | Your AWS region |
| `your-nexus-password` | Your Nexus admin password |
| `your-email` | Your notification email |
