# ☸️ EKS Cluster Setup

## Order of Setup:
1. Install kubectl
2. Install eksctl
3. Install Helm
4. Create kubeconfig
5. Associate OIDC Provider
6. Create Service Account
7. Install EBS CSI Driver
8. Install Ingress NGINX
9. Install Cert Manager
10. Apply K8s Manifests
